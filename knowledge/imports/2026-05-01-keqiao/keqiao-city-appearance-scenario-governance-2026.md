总体口径
1. 本资料用于柯桥区市容环境和城市管理事项的业务答复、线索分派、处置建议和知识库引用，不作为算法标签或感知规则说明。
2. 对群众投诉、巡查发现、热线转办、智慧城管采集的问题，应先区分事件类别、发生位置、责任主体、是否存在安全风险、是否需要跨部门协同。
3. 一般性市容环境问题按照“发现线索、核实事实、确定责任主体、派遣处置、复查反馈、评价归档”的闭环办理。
4. 涉及人员聚集、道路积水、火情烟雾、交通影响、河道防洪安全等风险事项时，应优先启动联动处置，避免只按普通市容案件处理。
5. 适用依据应按“国家行政法规、省级条例、绍兴市地方规定、柯桥区职责和工作机制”的顺序综合判断。

市容环境类事项

乱堆物堆料
1. 典型表现：在道路两侧、公共场地、商铺门前、广场周边、背街小巷、空闲地块周边堆放建筑材料、货物、杂物、经营物品或者废弃物，影响市容、通行或安全。
2. 依据口径：浙江省城市市容和环境卫生管理条例规定，任何单位和个人不得在街道两侧和公共场地堆放物料、搭建建筑物、构筑物或者其他设施；确因建设等特殊需要临时堆放的，应依法取得同意并办理相关手续。
3. 处置建议：现场核实堆放物品性质、数量、位置、占用范围、持续时间和责任主体；能当场清理的督促即时清理，不能当场清理的明确整改期限；涉及施工、商户经营、物业小区或空闲地块的，按责任区和主管事项分派。
4. 答复要点：该类问题不是简单“有没有物品”，关键看是否占用公共空间、是否影响通行和市容、是否具有合法临时占用手续、是否按要求限时恢复。

积存垃圾渣土
1. 典型表现：公共区域、道路边、工地周边、待建地块、河岸附近存在长期未清理的非生活垃圾、渣土、装修垃圾、淤泥、施工废弃物。
2. 依据口径：施工单位应及时清运、处置施工过程中产生的建筑垃圾；装修垃圾应堆放到指定地点；清理窨井淤泥产生的废弃物应及时清运处理，不得随意堆放。
3. 处置建议：先判断属于生活垃圾、建筑垃圾、装修垃圾、园林废弃物、河道淤泥还是偷倒渣土；固定位置、照片、数量和可能来源；建筑垃圾应衔接备案、清运、电子联单或执法线索，生活垃圾应转责任区保洁和分类收运。
4. 答复要点：积存垃圾渣土通常要同时说明“谁负责清、清到哪里、是否涉嫌违法倾倒、是否需要追溯来源”。

暴露垃圾
1. 典型表现：生活垃圾、废弃包装、果皮纸屑、杂物、废旧物品等裸露堆放在道路、广场、绿地、河岸、垃圾桶外或公共空间。
2. 依据口径：公民应维护城市环境卫生，不得乱扔废弃物、乱倒生活垃圾；环境卫生责任区责任人应保持责任区环境整洁，并对损害环境卫生的行为劝阻、制止和报告。
3. 处置建议：区分偶发抛撒、保洁不到位、垃圾桶容量不足、商户投放不规范、小区管理缺位等原因；先清除污染物，再追责责任主体或优化收运频次。
4. 答复要点：对群众问“这里垃圾没人管”，应优先说明属地、责任区、经营管理单位或环卫保洁单位的责任，并给出清理和复查路径。

垃圾满溢
1. 典型表现：垃圾箱、分类投放点、中转暂存点垃圾超过容器容量，出现箱外堆放、污水外溢、异味、病媒滋生风险。
2. 依据口径：生活垃圾分类投放、分类收集、分类运输、分类处置实行管理责任区和责任人制度；公共场所经营管理单位应按规定设置生活垃圾分类投放、分类收集设施并保持正常使用。
3. 处置建议：核实满溢发生时段、垃圾类型、桶点设置、清运频次和管理责任人；短期先调度清运和保洁，长期可调整投放时间、桶点容量、清运频次或宣传约束投放行为。
4. 答复要点：垃圾满溢不宜只答“清运即可”，还应分析是否存在投放高峰、桶点不足、清运计划不匹配或管理责任不到位。

绿地脏乱
1. 典型表现：公园绿地、道路绿化带、街头花坛、草坪内有生活垃圾、废弃包装、杂物、宠物粪便、修剪枝叶未清、绿化设施破损。
2. 依据口径：临街树木、绿篱、花坛、草坪应保持整洁美观；栽培、整修作业留下的渣土、枝叶等应及时清除；责任区责任人应保持环境卫生设施整洁完好。
3. 处置建议：区分日常保洁、绿化养护、游客乱扔、施工遗留和设施损坏；需要绿化养护单位、环卫保洁单位、属地镇街或公园管理单位协同的，应分别派遣。
4. 答复要点：绿地脏乱既涉及“垃圾清理”，也涉及“绿化养护”和“公共空间管理”，答复时应说明责任边界。

河道垃圾
1. 典型表现：河岸、湖岸、桥下、水边步道存在生活垃圾、建筑垃圾、枯枝杂物、油污垃圾或沿岸堆放废弃物。
2. 依据口径：绍兴市市容和环境卫生管理规定明确，水域管理单位应及时清理垃圾等水面漂浮物，保持水面清洁；浙江省河道管理条例禁止在河道管理范围内倾倒矿渣、石渣、泥土、泥浆、垃圾等废弃物。
3. 处置建议：先区分水面漂浮物、岸线垃圾、河道管理范围内倾倒、排口污染或涉水工程遗留；水面保洁派水域管理或保洁单位，岸上垃圾按属地和责任区处理，违法倾倒转执法调查。
4. 答复要点：河道垃圾要同时关注水面、岸线和河道管理范围，不能只按道路保洁问题处理。

河道漂浮物
1. 典型表现：水面存在塑料瓶、包装袋、泡沫、树枝、水草、水葫芦、浮萍、油污漂浮物等，影响水面清洁和水环境观感。
2. 依据口径：水域管理单位负有及时清理水面垃圾等漂浮物、保持水面清洁的职责；河道管理应兼顾防洪排涝、水生态环境和综合功能。
3. 处置建议：根据漂浮物类型判断来源，常规漂浮物安排打捞保洁，水草蓝藻类问题联动水利、农业农村或生态环境等部门，油污或异常污染应及时上报并溯源。
4. 答复要点：对“河道漂浮物很多”的问题，应说明打捞清理、来源排查、保洁频次和汛期大风天气后的加密巡查。

焚烧垃圾、树叶
1. 典型表现：在露天场所、垃圾收集容器内、绿化带旁、空闲地块、河岸边焚烧树叶、垃圾或者其他废弃物。
2. 依据口径：浙江省城市市容和环境卫生管理条例禁止在露天场所和垃圾收集容器内焚烧树叶、垃圾或者其他废弃物；违反的，应责令改正并可依法处罚。
3. 处置建议：发现明火或烟雾时先保障安全，必要时联动消防、公安、属地镇街；同步记录焚烧位置、焚烧物、责任人和影响范围；作业产生的枝叶应清运处理，不得现场焚烧。
4. 答复要点：焚烧类问题优先考虑火灾和空气污染风险，处置顺序是制止、灭火、清理、调查责任。

沿街晾挂
1. 典型表现：在道路、公园绿地、公共场所的护栏、电杆、树木、路牌、桥栏、公共设施上晾晒或吊挂衣物、布料、广告物、杂物。
2. 依据口径：浙江省城市市容和环境卫生管理条例规定，任何单位和个人不得在城市道路、公园绿地和其他公共场所的护栏、电杆、树木、路牌等公共设施上晾晒、吊挂衣物。
3. 处置建议：现场劝导并督促收回，反复发生的纳入重点点位巡查；涉及经营宣传、广告横幅或占用公共设施的，应进一步判断是否属于户外广告或设施占用问题。
4. 答复要点：沿街晾挂以劝导纠正和恢复市容为主，涉及安全隐患或拒不改正时再按规定处理。

占道撑伞
1. 典型表现：商贩、摊点、经营者在道路、人行道、广场、公共场地设置遮阳伞、棚伞、展架、桌椅等，形成占道经营、通行障碍或临时搭设。
2. 依据口径：任何单位和个人不得擅自占用城市人行道、桥梁、地下通道以及其他公共场所设摊经营、兜售物品；沿街经营者不得擅自超出门、窗进行店外经营、作业或者展示商品。
3. 处置建议：判断是否属于临时便民疏导点、是否超范围经营、是否影响通行和消防通道；对轻微情形先劝导规范，对拒不整改或反复占道的依法处理。
4. 答复要点：“撑伞”本身不是唯一判断点，关键看是否伴随设摊、经营、占道、遮挡、影响通行或存在安全隐患。

露天烧烤
1. 典型表现：道路两侧、公共场地、夜市周边、商铺门前、居民区附近露天设置烧烤炉具、桌椅、遮棚，占道经营并产生油烟、噪声、垃圾和消防隐患。
2. 依据口径：露天烧烤通常涉及占道经营、店外经营、市容环境卫生、餐饮油烟和消防安全等多重事项；沿街经营不得擅自超出门窗进行店外经营，公共场所不得擅自设摊经营。
3. 处置建议：先核实经营主体、经营地点、是否占道、是否产生油烟扰民、是否堵塞通道、是否有燃气和明火安全风险；一般问题由综合执法和属地规范经营，油烟超标或污染问题联动生态环境、市场监管等部门。
4. 答复要点：露天烧烤不能只按“餐饮”或“油烟”处理，应同时纳入市容秩序、环境卫生、消防安全和群众投诉综合研判。

宣传广告类事项

违规户外广告
1. 典型表现：未经同意设置大型户外广告，户外广告或招牌设施破损、污浊、腐蚀、陈旧、灯光显示不全，或设置位置影响市容、通行、采光、安全。
2. 依据口径：浙江省城市市容和环境卫生管理条例要求户外广告设施以及招牌、电子显示牌、灯箱、画廊、条幅、旗帜等户外设施符合城市容貌标准；大型户外广告设置应征得主管部门同意并依法办理审批手续；设置者应负责日常维护和保养。
3. 技术口径：城市户外广告和招牌设施技术标准适用于户外广告和招牌设施的规划、设计、施工、验收和运行管理，强调安全可靠、布局合理、视觉环境有序。
4. 处置建议：核实设施类型、设置主体、审批情况、位置、面积、结构安全和维护状态；未经同意、影响市容或存在安全隐患的，应责令整改、修复或拆除。
5. 答复要点：违规户外广告的判断不仅看“有没有广告”，还要看审批手续、设置标准、日常维护、安全风险和城市容貌影响。

突发事件类事项

人群聚集
1. 典型表现：广场、公园、商圈、学校周边、交通节点、活动现场等区域短时间内人员密集，可能造成拥堵、踩踏、秩序混乱、交通受阻或公共安全风险。
2. 依据口径：大型群众性活动安全管理遵循安全第一、预防为主，承办者负责、政府监管；预计参加人数达到法定规模的大型群众性活动应依法落实安全许可、安全工作方案、现场秩序维护和应急救援预案。
3. 处置建议：区分经许可活动、自发聚集、突发围观、商业促销、学校医院周边拥堵等情形；发现密度异常、通道受阻或秩序风险时，应及时联动公安、应急、交通、属地镇街和场所管理单位，先疏导控流再调查原因。
4. 答复要点：人群聚集不是普通市容问题，答复时应突出公共安全、属地联动、承办者或场所管理者责任。

道路积水
1. 典型表现：城市道路、下穿通道、低洼点、桥洞、地下空间出入口、施工路段出现积水，影响车辆和行人通行，严重时可能形成内涝风险。
2. 依据口径：城市排水防涝应急管理要求各地完善排水防涝应急机制，明确责任人，加强应急处置协同联动，保障城市安全度汛。
3. 处置建议：先判断积水深度、范围、是否危及车辆行人、是否涉及井盖移位、排水口堵塞、施工围挡、泵站运行或极端降雨；一般积水联动排水、市政和环卫清掏，严重内涝联动应急、交警、消防和属地管控。
4. 答复要点：道路积水应优先处理安全警示、交通疏导、排水抢险和原因排查，不能只作为普通保洁或路面问题答复。

其他城管事项

打包垃圾
1. 典型表现：商户、居民、临时经营者将袋装或打包好的生活垃圾、餐厨垃圾、废弃包装物放在人行道、绿化带、门前、桶外、墙角或公共区域。
2. 依据口径：生活垃圾应按规范分类投放、收集、运输和处置；管理责任区责任人应保持责任区环境整洁，发现损害环境卫生的行为应劝阻、制止并报告。
3. 处置建议：核实垃圾类型、投放时间、投放位置、责任主体和桶点容量；对商户门前打包垃圾，应结合“门前三包”或责任区管理督促规范投放；对居民小区应协调物业、业委会或社区优化投放提示和清运安排。
4. 答复要点：打包垃圾和暴露垃圾的区别在于前者通常已袋装或集中打包，但只要未按规定投放、造成桶外堆放或环境影响，仍应纳入环境卫生问题处置。

常见问答口径
1. 问：乱堆物堆料应该怎么处理？
答：先核实是否占用道路两侧或公共场地、是否影响通行和市容、是否有临时占用手续，再确定责任主体并督促清理；涉及施工、商户、物业或空闲地块的，分别转对应责任单位处置。
2. 问：垃圾满溢是不是只要通知环卫清运？
答：短期要及时清运和保洁，长期还要核实桶点容量、投放高峰、清运频次和责任区管理是否匹配。
3. 问：河道漂浮物归谁处理？
答：水面垃圾和漂浮物一般由水域管理或河道保洁责任单位及时清理；岸线垃圾、违法倾倒、污染排口等问题应按责任主体和主管职责分派。
4. 问：道路积水要不要走城管派单？
答：可以作为城市管理线索登记，但处置上应优先联动排水、市政、交警、应急等单位，先保障通行安全和排水抢险。
5. 问：露天烧烤怎么归类？
答：露天烧烤往往同时涉及占道经营、店外经营、油烟扰民、垃圾污染和消防安全，应根据现场事实综合分派和处置。
